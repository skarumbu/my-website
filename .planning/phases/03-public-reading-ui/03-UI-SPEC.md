## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, tag padding |
| sm | 8px | Between description and date, row internal gaps |
| md | 16px | Default element spacing, paragraph margins |
| lg | 24px | Section padding, between list rows |
| xl | 32px | Layout horizontal padding (mobile) |
| 2xl | 48px | Major section breaks, page top padding |
| 3xl | 64px | Page-level bottom clearance |

Exceptions:
- List row vertical padding: 20px top/bottom (splits md and lg — matches ideas-card padding precedent)
- Wide-screen horizontal padding on reader column: 40px left/right (matches `.ideas-content` at `40px`)
- Reader column max-width: 68ch (typography-relative, not px — required by D-04)

Source: Derived from existing patterns in `ideas.css` (`.ideas-content { padding: 24px 40px }`, `.ideas-card { padding: 20px 22px }`).

## Typography

All sizes and weights match the existing DM Sans + Fraunces system already in use site-wide.

Declared sizes: exactly 4 (13px, 16px, 22px, 48px).
Declared weights: exactly 2 (400 regular, 600 semibold).

| Role | Size | Weight | Line Height | Font | Usage |
|------|------|--------|-------------|------|-------|
| Small / label | 13px | 400 | 1.4 | DM Sans | Date display on list rows and reader header, "Back to Writing" link, inline code |
| Body | 16px | 400 | 1.6 | DM Sans | Post prose paragraphs, description text on list rows, H3 headings in markdown body |
| Heading | 22px | 600 (Fraunces variation) | 1.25 | Fraunces | Post title on list row, post title on reader header, H2 headings in markdown body |
| Display | 48px | 400 | 0.95 | Fraunces | "Writing" H1 on /posts page |

Additional markdown body typography (rendered by react-markdown inside reader):

| Element | Size | Weight | Line Height | Font |
|---------|------|--------|-------------|------|
| `h1` in body | 22px | 600 | 1.25 | Fraunces |
| `h2` in body | 22px | 600 | 1.25 | Fraunces |
| `h3` in body | 16px | 600 | 1.3 | DM Sans |
| `p` | 16px | 400 | 1.6 | DM Sans |
| `code` inline | 13px | 400 | inherit | monospace (system: `ui-monospace, 'Cascadia Code', 'Source Code Pro', monospace`) |
| `pre` / code block | 13px | 400 | 1.5 | monospace |

Source: Heading size from `ideas.css` `.ideas-card-title { font-size: 20px }` and `.ideas-heading { clamp(48px, 6vw, 72px) }` — page heading uses Fraunces at 48px to match Ideas page heading pattern. Body 16px / 1.6 is standard editorial line-height, consistent with `.ideas-card-body { font: 400 14px/1.55 }` trend, stepped up to 16px for long-form reading comfort.

## Color

All values taken from existing `:root` CSS variables defined in `src/styling/home.css` and `src/styling/ideas.css`.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#FFEFD9` (`--bg-1`) | Page background on /posts and /posts/:slug — same warm cream as all other pages |
| Secondary (30%) | `#FFFFFF` / `rgba(255,255,255,0.85)` | List row hover background, reader column surface, code block background (slightly darker: `rgba(58,36,24,0.04)`) |
| Accent (10%) | `#E5533C` (`--accent`) | Nav link active state, inline hyperlinks within post body, and error text |
| Ink / text | `#3A2418` (`--ink`) | All body text, post titles, page heading |
| Ink secondary | `#6B4A3A` (`--ink-2`) | Post description on list row, reader prose (slightly muted) |
| Ink tertiary | `#A38876` (`--ink-3`) | Date display, back-link muted text |
| Border / divider | `#F0D7B6` (`--line`) | List row dividers, reader column rule below header |
| Error | `#E5533C` (`--accent`) | `.posts-error` inline error text (matches existing `ideas-error` pattern) |

Accent reserved for: nav link active state, inline hyperlinks within post body, and error text.

Source: All hex values copied directly from `src/styling/home.css` `:root` block.

**Tone note (from CONTEXT.md specifics):** Posts pages intentionally avoid the candy-pastel backgrounds (`--c-coral`, `--c-butter`, etc.) used on home page cards. The dominant surface stays plain `#FFEFD9` with white secondaries — calmer, editorial feel vs. the playful home page.

## Component Inventory

### Posts.tsx — /posts list page

**Page shell:**
- Class: `.posts-page` — `min-height: 100vh; background: var(--bg-1); font-family: 'DM Sans'; color: var(--ink);`
- Uses `.main` wrapper from `src/styling/main.css` for base layout

**Page heading:**
- `<h1 className="posts-heading">Writing</h1>`
- Fraunces, 48px, weight 400, line-height 0.95, letter-spacing -0.03em, color `--ink`
- Margin: 48px top, 32px bottom (before list)
- Source: CONTEXT.md D-02, D-03 (label is "Writing")

**Post list:**
- Class: `.posts-list` — `list-style: none; margin: 0; padding: 0; max-width: 720px; width: 100%;`
- Each item: `.posts-row` — full-width, no card background in default state
- Row layout: CSS Grid `grid-template-columns: 1fr auto` for title+date on first row, then description spanning full width below
- Row padding: 20px top/bottom, 0 horizontal
- Row separator: `border-bottom: 1px solid var(--line)`
- Row hover: `background: rgba(255,255,255,0.6)` with `border-radius: 8px; padding-left: 12px; transition: background 180ms`
- Source: CONTEXT.md D-01

**Inside each row:**
- `.posts-row-title` — Fraunces, 22px, weight 600 (variation settings matching `ideas-card-title`), `color: var(--ink)`, left-aligned
- `.posts-row-date` — DM Sans, 13px, weight 400, `color: var(--ink-3)`, right-aligned, `white-space: nowrap`
- `.posts-row-desc` — DM Sans, 16px, weight 400, `color: var(--ink-2)`, `line-height: 1.5`, `margin-top: 4px`, grid-column span 2

**Loading state:**
- Render `<Spinner />` from `src/components/Spinner.tsx` centered in page
- Source: CONTEXT.md Claude's Discretion + Spinner.tsx pattern

**Error state:**
- `<p className="posts-error">{error}</p>` — DM Sans, 13px, color `var(--accent)`, margin `0 0 12px`
- Copy: "Could not load posts. Check your connection and refresh."
- Source: CONTEXT.md (inline `p.posts-error` pattern), `ideas.css` `.ideas-error` precedent

**Empty state:**
- Shown when API returns empty array (not an error)
- `<p className="posts-empty">No posts yet. Check back soon.</p>` — centered, DM Sans 16px, color `var(--ink-2)`

**Nav integration:**
- Standard `<NavBar />` from `src/components/nav-bar.tsx` at top of page
- "Writing" link added to nav-bar.tsx linking to `/posts`
- Source: CONTEXT.md D-09

### PostReader.tsx — /posts/:slug reader page

**Page shell:**
- Class: `.post-reader-page` — `min-height: 100vh; background: var(--bg-1); font-family: 'DM Sans'; color: var(--ink);`

- Class: `.post-reader-content` — `max-width: 68ch; width: 100%; margin: 0 auto; padding: 48px 40px 64px;`

- DM Sans, 13px, weight 400, color `var(--ink-3)`, no underline, hover: `color: var(--ink)`
- Margin-bottom: 32px
- Source: CONTEXT.md D-08, READ-05

**Post header (above body):**
- `.post-reader-header` — margin-bottom 32px, padding-bottom 24px, border-bottom `1px solid var(--line)`
- `.post-reader-title` — Fraunces, 22px, weight 600, line-height 1.25, color `var(--ink)`, margin 0 0 8px
- `.post-reader-date` — DM Sans, 13px, weight 400, color `var(--ink-3)`, margin 0 0 12px
- `.post-reader-desc` — DM Sans, 16px, weight 400, color `var(--ink-2)`, line-height 1.5, margin 0
- Source: CONTEXT.md D-05

**Markdown body:**
- Class: `.post-reader-body` — contains all react-markdown output
- Rendered via `react-markdown` — no `dangerouslySetInnerHTML` (SEC-01, D-11)
- Prose styles applied to `.post-reader-body` children:
  - `p`: 16px, weight 400, line-height 1.6, color `var(--ink)`, margin 0 0 16px
  - `h1`: Fraunces 22px, weight 600, line-height 1.25, margin 32px 0 12px
  - `h2`: Fraunces 22px, weight 600, line-height 1.25, margin 28px 0 12px
  - `h3`: DM Sans 16px, weight 600, line-height 1.3, margin 24px 0 8px
  - `a`: color `var(--accent)`, no underline default, underline on hover
  - `strong`: weight 600, color `var(--ink)`
  - `em`: font-style italic
  - `ul`, `ol`: margin 0 0 16px, padding-left 24px
  - `li`: margin 0 0 4px, line-height 1.6
  - `code` inline: `background: rgba(58,36,24,0.06); border-radius: 4px; padding: 4px 8px; font-family: ui-monospace, 'Cascadia Code', 'Source Code Pro', monospace; font-size: 13px; color: var(--ink);`
  - `pre`: `background: rgba(58,36,24,0.05); border: 1px solid var(--line); border-radius: 8px; padding: 16px; overflow-x: auto; margin: 0 0 16px;`
  - `pre code`: `background: none; padding: 0; border-radius: 0; font-size: 13px; line-height: 1.5; color: var(--ink);`
  - `blockquote`: `border-left: 3px solid var(--line); padding-left: 16px; color: var(--ink-2); margin: 0 0 16px;`
- Source: CONTEXT.md D-06 (code blocks: monospace, darker background, padding — no syntax highlighting)

**Loading state:** `<Spinner />` centered while fetching post content

**Error state:** `<p className="posts-error">Could not load this post. It may have been moved or removed. Return to <a href="/posts">Writing</a>.</p>`

**Nav:** Standard `<NavBar />` at top

## Interaction Contract

### Post list row
- Entire row is a clickable link (`<a>`) to `/posts/:slug`
- Hover: subtle background fill `rgba(255,255,255,0.6)`, `border-radius: 8px`, `padding-left: 12px`, transition 180ms
- Cursor: pointer
- No separate "Read" button — the row itself is the CTA

### Back link
- Click navigates to `/posts`
- Hover: color changes from `var(--ink-3)` to `var(--ink)`, transition 180ms
- No animation

### Home page card (Writing)
- Same hover as all home page cards: `translateY(-6px) rotate(-1deg)`, box-shadow deepens
- Transition: `260ms cubic-bezier(.34,1.56,.64,1)` (matches existing card pattern exactly)

### Loading → content transition
- No animation — content replaces spinner immediately on fetch completion
- Matches the pattern used in MomentumFinder and Ideas

### Nav link ("Writing")
- Active state: dark pill background `var(--ink)` with text `#FFEFD9` (matches existing `.home-nav-links a.active` and nav-bar `.button` active conventions)

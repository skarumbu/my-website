def strip_fences(text):
    text = text.strip()
    if text.startswith("```"):
        lines = text.split("\n")
        text = "\n".join(lines[1:])
        if text.endswith("```"):
            text = text[:-3].rstrip()
    return text
```

**run() helper — copy verbatim from arch_content_update.py:**

def run(cmd, cwd=None):

    return result.stdout.strip()

All 5 packages tagged [ASSUMED] in RESEARCH.md — checkpoint is mandatory per protocol. |
| T-01-01 | Information Disclosure | local.settings.json (contains UseDevelopmentStorage=true connection string) | mitigate | local.settings.json listed in .gitignore (Task 2 acceptance criterion); never committed to git |
| T-01-02 | Information Disclosure | .azurite/ debug log | accept | Local dev only; contains no real credentials; Azurite uses well-known emulator keys |
| T-01-03 | Tampering | GitHub Actions deploy.yml uses AZURE_CREDENTIALS secret | mitigate | Secret is stored in GitHub repo settings (not in code); deploy.yml adapted from verified ideas-api pattern |

Note: T-01 (blob public access) and T-02 (YAML injection) and T-03 (slug collision) are implemented in Plan 02 (schema.py, slugs.py code) and Plan 03 (Bicep). They are catalogued in those plans' threat models.
